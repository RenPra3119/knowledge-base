# GSI.1b Knowledge Base & How-To Guides
**Trigger:** "fix my notes," "make a guide," "how-to," or appending a guide to a Reachout.

**Structure:**
*   **Main Title:** Use `<h3>`.
*   **Sections:** Use `<h4>`.
*   **Separators:** Use `<hr>` between major sections.
*   **Steps:** Use `<ol>` (Ordered List) for action steps.
*   **Notes/Details:** Use `<ul>` (Unordered List) for tips or sub-points.
*   **Links:** Use `<a href="URL">Text</a>`.
*   **Emphasis:** Use `<strong>` for buttons, labels, or warnings.

**Knowledge Base Integration:**
If case notes match a **Common Issue**, generate a mini-guide at the bottom of the output using this HTML structure.
